Detergent Concentration Tool

Note: Chlorinated product – add 3 drops Sodium Thiosulphate in Step 2.
Concentration 0.5 g/L1.0 g/L1.5 g/L 2.0 g/L3.0 g/L6.0 g/L
Drops

Instructions

  1. Fill the test tube to the 5 ml mark with water from the wash tank.
  2. Add 3 drops of Sodium Thiosulphate.
  3. Add 1 drop of Indicator Reagent — solution turns pink.
  4. Select the correct acid (ACID 3 is 4× ACID 1).
  5. Add acid drop by drop, swirl each time.
  6. Stop when colour turns colourless.
  7. Count drops → read table.
